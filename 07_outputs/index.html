For this assignment I desided to resucitate my project from the previous week, the ultrasonic distance sensor that lighted up a green LED when an object was less than 10 cm away from the sensor. At the beginning I faced some difficulties because I forgot to ground the oscilloscope itself. The smallest interval is only 110 ms which indicates that the time domain is really in the seconds rather than the miliseconds. Additionally, we can observe the voltage around 3V when the LED turns on and around 0V when it is off. This is consistent with the circuit.
